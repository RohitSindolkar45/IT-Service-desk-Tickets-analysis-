IT Service Desk Ticket Analysis

Project Overview:

This project analyzes IT service desk ticket data to evaluate SLA performance, ticket resolution efficiency, and service desk operations.
The analysis was performed using Excel for data cleaning, MySQL for data analysis, and Power BI for interactive dashboard visualization.

Tools & Technologies:

Excel – Data cleaning and preprocessing
MySQL – Data analysis using SQL queries
Power BI – Data modeling and interactive dashboard creation

Dataset:

The dataset contains IT service desk ticket information including:

Ticket ID
Agent ID
Issue Category
Priority
SLA (Service Level Agreement)
Resolution Time
SLA Breach Status

Two datasets were used:

Ticket Data – Contains ticket details and performance metrics
Agent Data – Contains agent-related information

Project Workflow:

Data Cleaning (Excel):

Removed duplicates and handled missing values
Standardized column formats
Prepared data for database import

Data Analysis (MySQL):

Performed ticket volume analysis
Identified SLA breaches
Analyzed agent performance
Evaluated issue category trends

Dashboard Development (Power BI):

Created data model using fact and dimension tables
Built interactive visuals and KPI cards
Developed service desk performance dashboards

Key Insights:

Identified agents contributing most to SLA breaches
Analyzed correlation between SLA targets and resolution time
Determined issue categories with longest resolution times
Monitored ticket distribution across priorities and teams

Key Dashboard Metrics:

Total Tickets
SLA Breach Tickets
On-Time Resolution %
High Priority Tickets
Average Resolution Time

Project Outcome:

This project demonstrates the use of data analytics techniques to monitor IT service desk performance and identify operational inefficiencies using real-world analytical tools.
